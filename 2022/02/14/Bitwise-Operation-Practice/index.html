Bitwise AND of Numbers Range

  • Given two integers left and right that represent the range [left, right], return the bitwise AND of all numbers in this range, inclusive.

Constraints

  • 0 <= left <= right <= 231 - 1

Approach 1

  • Traverse all element between left and right and do bitwise AND operation.
  • However, this approach is very inefficient.

Approach 2

  • Note that if this range contains \(01111,10000\) number of \(1\)'s of the first and \(0\)'s of the second are equal, then the result is \(0\).
  • Also, if the upper and lower bound have the same first \(x\) digits, then the result also have the same first \(x\) digits.
  • For the rest digits, the result will be all zero since there must be some step from \(011\) to \(100\)
